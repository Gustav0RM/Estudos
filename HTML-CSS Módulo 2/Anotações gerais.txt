		<IMAGENS>
			Para carregar uma imagem que está dentro da mesma pasta do seu arquivo, se usa :
			<img src = "ctrl + espaço e seleciona a imagem"
			alt = "descrição da imagem">
			Para carregar uma imagem que está dentro de uma sub-pasta dentro da pasta do arquivo, se usa o mesmo comando buscando pelo ctrl + espaço.
			E para carregar uma imagem que está em outro local da internet, basta copiar o link da foto e colar no src.
	
		</IMAGENS>
	
	
	
		<LINKS>
			Para criar um link de um site externo basta utilizar a tag <a href="Inserir o link aqui" target="_blank" rel="external">Inserir aqui frase que deseja conter o link </a>
	
			* target="_blank" serve para abrir o link em uma nova aba
			* rel="external" serve para indicar que o site que está no link não faz parte do site atual
			* rel="next" serve para indicar que o link é para proxima parte do site atual
			* rel="prev" serve para indicar que o link é para a parte anterior do site atual
			* rel="author" serve para indicar um link para o site do autor do artigo atual
			* rel="nofollow" serve para indicar que é um link para um site pago
	
		</LINKS>
	
	
	
		<MEDIA>

						**ORDEM DAS TAGS**
			<picture>
					<source>  IMG PEQUENA </source>
					<source> IMG MEDIA  </source>
					<img>  IMG GRANDE   <img>
			</picture>
	
	
		  *VIDEOS para adicionar um player de video do youtube é usada tag <iframe> com atributos src= link do youtube, title= nome do video, frameborder= borda do frame, allow= autorizar funcionalidades do player, allowfullscreen permitir tela cheia.
	    </MEDIA>
